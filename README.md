# hello-react-2025

2025년 "리액트를 다루는 기술" 도서를 학습하며 실습하는 학습 프로젝트입니다.

## 기술 스택

- **React 19.1.1** - 사용자 인터페이스 라이브러리
- **TypeScript 5.8.3** - 타입 안전성을 제공하는 JavaScript 확장
- **Vite 7.1.6** - 빠른 빌드 도구 및 개발 서버
- **ESLint** - 코드 품질 관리

## 시작하기

### 개발 서버 실행
```bash
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```

### 코드 린팅
```bash
npm run lint
```

### 빌드 미리보기
```bash
npm run preview
```

## 프로젝트 구조

```
src/
├── App.tsx          # 메인 애플리케이션 컴포넌트
├── main.tsx         # 애플리케이션 진입점
├── index.css        # 글로벌 스타일
├── App.css          # App 컴포넌트 스타일
└── assets/          # 정적 자산 (이미지, 아이콘 등)
```

## 학습 목표

이 프로젝트는 다음과 같은 React 개념들을 학습하고 실습하기 위해 설계되었습니다:

- React 컴포넌트 기초
- 상태(State) 관리
- 이벤트 처리
- TypeScript와 React 통합
- 모던 개발 환경 (Vite, HMR)

## 개발 환경

- Node.js 18+ 권장
- npm 또는 yarn 패키지 매니저
